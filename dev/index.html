MyPkg54.jl

A Julia package for solving my problems.

Installation

Run the following command in the Julia REPL or a notebook:

julia> import Pkg; Pkg.add(url="https://github.com/ohno/MyPkg54.jl.git")

Quick Start

After installation, run the following to load the package and verify it works:

julia> import MyPkg54; MyPkg54.hello()"Hello, World!"
